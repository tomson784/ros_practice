## このサイトの編集方法

```
git clone https://github.com/tomson784/ros_practice.git
```

ディレクトリ構成は以下のようになっており`src`の部分に編集するファイルが入っています．
```
(root)/
    ├─ .gitignore
    ├─ book.toml
    ├─ docker-compose.yml
    ├─ Dockerfile
    ├─ LICENSE
    ├─ README.md
    ├─.github
    │  └─workflows
    │          main.yml
    └─src
        ├─ README.md
        ├─ SUMMARY.md
        ├─    :
        ├─    :
        ├─    :
```
